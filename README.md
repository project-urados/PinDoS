# PinDoS
PinDoS is an aggressive DoS system that launches a full-scale DoS attack on its target.
How to use:

``
cd Pindos
python3 pindos.py
``
Running pindos.py will drop you into a simple GUI where you enter the target IP and Port, the number of DoS bots, and the number of 'missiles' to use (>100000 recommended).
