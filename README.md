# PinDoS
PinDoS is an aggressive DoS system that launches a full-scale DoS attack on its target.
How to use:

```bash
git clone https://github.com/project-urados/PinDoS.git
cd Pindos
python3 pindos.py
```

Running "pindos.py" will drop you into a simple GUI where you enter the target IP and Port, the number of DoS bots (processes), and the number of 'missiles' (count) to use (>100000 recommended).
You can get a target IP by running

```bash
python3 url2ip.py example.com

```
