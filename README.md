# PinDoS
PinDoS is an aggressive DDoS system that launches a DDoS attack on its target.
How to use:

```bash
git clone https://github.com/project-urados/PinDoS.git
cd Pindos
python3 pindos.py
```

Running "pindos.py" will drop you into a simple GUI where you enter the target IP and Port, the number of DoS bots (processes), and the number of 'missiles' (count) to use (>100000 recommended).
You can get a target IP by running

```bash
python3 url2ip.py example.com
# Example website ^^^^^^^^^^^
```

## Introducing **PinDoS**

**PinDoS** is a *Distributed Denial of Service* tool that can put a heavy load on HTTPS servers and stop them from functioning normally by exhausting their resource pool.
